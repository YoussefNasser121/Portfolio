# My Personal Portfolio

Hey! Welcome to my portfolio website. It's a place where I showcase my projects, share what I've been working on, and make it easy for people to get in touch with me.

Built with React on the frontend and Node.js/Express on the backend. The whole thing is responsive, so it looks good on your phone, tablet, or laptop.

## What's Inside

- **Mobile-First Design** - Works smoothly on any device
- **Live GitHub Projects** - My repos automatically show up here
- **contact me** - Contact form that actually sends messages to my database
- **Skills List** - All the tech stuff I'm comfortable working with
- **Dark Theme** - Easy on the eyes (especially at 3 AM when coding)
- **Responsive Navbar** - Hamburger menu that actually works on mobile

## What I Used

**Frontend Side:**
- React (for building the UI)
- React Router (for page navigation)
- Bootstrap (makes everything look clean)
- Custom CSS for that extra flair

**Backend Side:**
- Node.js & Express (the server stuff)
- MongoDB (where your messages go)
- Mongoose (helps talk to the database)
